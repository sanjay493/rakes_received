### Best Performance
```

1. Calculate average transit time
2. Find period with minimum average

4. Store rake count from that period

### Outlier Detection
```
1. Calculate Q1 (25th percentile) and Q3 (75th percentile)
2. Calculate IQR = Q3 - Q1
3. Lower bound = Q1 - 1.5 Ã— IQR
4. Upper bound = Q3 + 1.5 Ã— IQR
5. Flag values outside [lower, upper] as outliers
