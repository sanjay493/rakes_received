## Design Principles

1. **Contrast**: Each analysis type uses distinct color combinations for easy identification
2. **Accessibility**: High contrast between lines and bars for better readability
3. **Professional**: Material Design inspired colors
4. **Consistency**: 
   - Transit time always uses solid, vibrant colors
   - Rake count uses semi-transparent bars with matching axis colors
   - Text labels match their respective graph elements

## Marker Specifications
- **Size**: 10px for clear visibility
- **Line**: 2px white border around markers for definition
- **Text Size**: 10px for labels
- **Text Position**: 
  - Transit time: "top center"
  - Rake count: "outside" (above bars)
